# Index Crawler

Example usage: `node main.js --url=http://www.jweemaes.nl/ebooks/2015/ --dir=/home/steve/Downloads/ebooks`

## Dependencies:
- NodeJS
- Packages via NPM, see `package.json`.
- wget
- A shell (Bash?)

## Warning!
This script uses a shell and wget.
